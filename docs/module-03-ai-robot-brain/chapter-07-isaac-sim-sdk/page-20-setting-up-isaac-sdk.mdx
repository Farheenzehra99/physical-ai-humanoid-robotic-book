## Introduction

**NVIDIA Isaac Sim** is a photorealistic, GPU-accelerated robotics simulation platform built on NVIDIA Omniverse. It enables sim-to-real transfer, allowing you to train and test humanoid robots in virtual environments before deploying to physical hardware.

:::note Historical Context
The term "Isaac SDK" historically referred to NVIDIA's robotics software toolkit (deprecated in 2023). This guide focuses on **Isaac Sim 2024.1+**, the current robotics simulation platform. When we refer to "Isaac SDK setup," we mean configuring Isaac Sim for robot development.
:::

### Software Requirements

Before proceeding, verify you have:

1. **NVIDIA GPU Driver 525.60.13 or higher**
   - Download: [NVIDIA Driver Downloads](https://www.nvidia.com/download/index.aspx)
   - Verify version:
     ```bash
     nvidia-smi
     ```

2. **CUDA Toolkit 12.1 or higher** (Ubuntu only)
   - Download: [CUDA Toolkit Archive](https://developer.nvidia.com/cuda-downloads)
   - Verify version:
     ```bash
     nvcc --version
     ```

3. **Python 3.10** (recommended for Isaac Sim 2024.1)
   - Ubuntu:
     ```bash
     sudo apt install python3.10 python3.10-venv python3-pip
     ```
   - Windows: [Python.org Downloads](https://www.python.org/downloads/)

4. **At least 50GB free disk space**

:::warning Driver Compatibility
Isaac Sim requires **RTX-capable NVIDIA GPUs** (GTX 10-series or newer). Older GPUs (GTX 900-series) may encounter compatibility issues with ray tracing features.
:::

if __name__ == "__main__":
