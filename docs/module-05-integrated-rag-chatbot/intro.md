# Integrated RAG Chatbot

Welcome to the Integrated RAG (Retrieval Augmented Generation) Chatbot for Interactive Technical Books. This feature allows readers to ask questions about book content and receive accurate answers with citations.

## Overview

The Integrated RAG Chatbot provides two modes of operation:

1. **Normal Mode**: Query the entire book content using a RAG pipeline with Qdrant vector database and Neon metadata store
2. **Selected Text Mode**: Query only the highlighted/selected text in the book

## Key Features

- Accurate answers sourced directly from book content
- Proper citations with chapter, page, and text snippets
- Two operation modes for different use cases
- Structured JSON responses with confidence scores
- Integration with Claude MCP for embeddings

## Getting Started

To start using the RAG Chatbot, navigate to any book page and use the chat interface that appears alongside the content.
